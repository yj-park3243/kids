```xml
<meta-data android:name="com.naver.maps.map.CLIENT_ID" android:value="539desbv96"/>
```

**iOS** (`ios/Runner/Info.plist`)
```xml
<key>NMFClientId</key>
<string>539desbv96</string>
```

### Firebase 설정

- `google-services.json` (Android) → kids Firebase 프로젝트에서 다운로드

## 7. 배포 방식

| 항목 | 방식 |
|------|------|
| 서버 배포 | 로컬 `server/deploy.sh` → rsync로 EC2 전송 → 원격에서 `npm install` + `npm run build` → `pm2 restart` |
| 앱 배포 | `app/deploy.sh` → `flutter build ipa/appbundle` → TestFlight / Google Play 업로드 |
| 관리자 대시보드 배포 | `admin/` Vite 빌드(`npm run build`) → 정적 호스팅(S3 + CloudFront 등) 예정 |
| DB 마이그레이션 | TypeORM migration:run |
