# 프로젝트 구조

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 모바일 앱 | Flutter (Dart) |
| 상태관리 | Riverpod |
| 라우팅 | go_router |
| HTTP | Dio |
| 관리자 대시보드 | React 19 + Vite + Ant Design + Recharts (`admin/`) |
| 서버 | NestJS (TypeScript) |
| ORM | TypeORM |
| DB | PostgreSQL |
| 실시간 채팅 | NestJS WebSocket Gateway (Socket.IO, namespace `/chat`) |
| 채팅 저장 | PostgreSQL (`chat_message` 테이블, Room.id == 채팅방 ID) |
| 인증 | JWT (passport-jwt) + `TokenService` (type/iss/jti 클레임) |
| 소셜 로그인 | Apple, Google (동작) / Kakao (서버는 구현, 앱 SDK 미연결) |
| 이메일 로그인 | bcrypt + JWT |
| 본인 인증 | **KCP V2** (사이트코드 `ALQ1Q`) — `auth/kcp` 모듈에서 HTML form 발급 + callback 복호화 |
| 푸시 | Firebase Cloud Messaging + APNs key |
| 파일 저장소 | AWS S3 (`kids-uploads-518`, presigned URL) |
| 지도 | 네이버 지도 SDK + Geocoding API (room 모듈에 통합) |
| 공유 | Kakao Flutter SDK (`kakao_flutter_sdk_share`) — Custom Template 기반 방 공유 |
| 딥링크 | `app_links` (Flutter) — `kids://room/:id` + Android App Link / iOS Universal Link |
| 캐시 | (미구현, 현재 in-memory. 이후 Redis 도입 검토) |

### 기술 스택
- **React 19 + TypeScript + Vite 8** — 빠른 HMR
- **Ant Design 6** — 폼, 테이블, Message API
- **Recharts 3** — 대시보드 통계 차트
