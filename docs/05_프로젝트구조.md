# 프로젝트 구조

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 모바일 앱 | Flutter (Dart) |
| 상태관리 | Riverpod |
| 라우팅 | go_router |
| HTTP | Dio |
| 관리자 대시보드 | React 19 + Vite + Ant Design + Recharts (`admin/`) |
| 서버 | NestJS (TypeScript) |
| ORM | TypeORM |
| DB | PostgreSQL |
| 실시간 채팅 | NestJS WebSocket Gateway (Socket.IO, namespace `/chat`) |
| 채팅 저장 | PostgreSQL (`chat_message` 테이블, Room.id == 채팅방 ID) |
| 인증 | JWT (passport-jwt) |
| 소셜 로그인 | 카카오, Apple, Google |
| 이메일 로그인 | bcrypt + JWT |
| 본인 인증 | 핸드폰 SMS OTP (발송 업체 연동 예정) |
| 푸시 | Firebase Cloud Messaging |
| 파일 저장소 | AWS S3 (`kids-uploads-518`, presigned URL) |
| 지도 | 네이버 지도 SDK + Geocoding API (room 모듈에 통합) |
| 캐시 | (미구현, 현재 in-memory. 이후 Redis 도입 검토) |

### 기술 스택
- **React 19 + TypeScript + Vite 8** — 빠른 HMR
- **Ant Design 6** — 폼, 테이블, Message API
- **Recharts 3** — 대시보드 통계 차트
- **React Router DOM 7** — 라우팅 및 ProtectedRoute 구현
- **Axios** — API 클라이언트 (`{ success, data }` 자동 언래핑)
- **Playwright** — E2E 스모크 테스트

### Docker Compose (로컬 개발)

```yaml
# docker-compose.yml
version: '3.8'
services:
  postgres:
    image: postgres:16
    environment:
      POSTGRES_DB: kids
      POSTGRES_USER: kids
      POSTGRES_PASSWORD: kids1234
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

volumes:
  pgdata:
```
