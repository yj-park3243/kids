# 공동육아 플랫폼 기획서

## 서비스명 (가칭): 같이크자

> "우리 아이와 비슷한 또래 친구를 동네에서 쉽게 만나는 육아 모임 플랫폼"

---

## 1. 서비스 개요

### 1.1 문제 정의

- 비슷한 개월수 아이를 키우는 부모들이 주변에서 육아 친구를 찾기 어려움
- 놀이터/키즈카페에서 우연히 만나는 것 외에 체계적인 연결 수단이 없음
- 기존 맘카페는 온라인 커뮤니티일 뿐, 실제 만남으로 이어지기 어려움

### 1.2 해결 방안

- **지역 + 날짜/시간 + 개월수** 기반으로 육아 모임 "방"을 개설
- 조건이 맞는 부모가 방을 검색하고 참여 신청
- **자유 입장** 또는 **방장 수락** 방식으로 운영

### 1.3 타겟 유저

| 구분 | 설명 |
|------|------|
| 주 타겟 | 0~36개월 아이를 키우는 부모 (엄마/아빠) |
| 보조 타겟 | 37~72개월(미취학) 아이를 키우는 부모 |

## 2. 핵심 기능

### 2.1 방 (모임) 시스템

핵심 플로우: **방 생성 → 검색/발견 → 참여 신청 → 모임 진행 → 후기**

#### 방 생성

| 필드 | 필수 | 설명 |
|------|------|------|
| 제목 | O | 모임 제목 |
| 설명 | O | 모임 소개 |
| 지역 | O | 시/구/동 단위 선택 |
| 날짜 | O | 모임 날짜 |
| 시작 시간 | O | 모임 시작 시간 |
| 종료 시간 | X | 예상 종료 시간 |
| 대상 개월수 | O | 범위 지정 (예: 6~12개월) |
| 장소 유형 | O | 놀이터 / 키즈카페 / 파티룸 / 공원 / 기타 |
| 장소 상세 | X | 구체적 장소명, 주소 |
| 최대 인원 | O | 2~10명 |
| 입장 방식 | O | 자유 입장 / 승인 필요 |
| 비용 | X | 무료 / 1인당 금액 |
| 태그 | X | #이유식교류 #산책 #놀이 등 |

#### 방 상태

```
모집중 → 모집완료 → 진행중 → 완료 → (자동 종료)
         ↘ 취소
```

#### 입장 방식

- **자유 입장**: 신청 즉시 참여 확정
- **승인 필요**: 방장이 프로필 확인 후 수락/거절

### 2.2 검색 및 필터

| 필터 | 설명 |
|------|------|
| 지역 | 내 동네 / 인근 동네 |
| 날짜 | 오늘 / 내일 / 이번 주 / 날짜 선택 |
| 개월수 | 내 아이 개월수 기준 자동 필터 |
| 장소 유형 | 놀이터 / 키즈카페 / 파티룸 등 |
| 비용 | 무료만 / 전체 |

> **규칙**: 약속 날짜가 지난 방은 목록에서 자동 제외 (조회 불가). 본인이 참여한 지난 모임은 마이페이지 > 지난 모임에서만 확인 가능.

### 2.3 네이버 지도 뷰

방 목록을 지도 위에서 탐색하는 기능

| 줌 레벨 | 표시 방식 | 설명 |
|---------|-----------|------|
| 축소 (광역) | 클러스터 (원형 뱃지) | 동네별 모집중인 방 개수를 원 안에 숫자로 표시 |
| 확대 (상세) | 핀 마커 | 개별 방의 약속 장소에 핀 표시, 탭하면 방 요약 팝업 |

- 네이버 지도 SDK (naver_map_plugin) 사용
- 방 생성 시 장소 좌표(위도/경도)를 네이버 Geocoding API로 변환하여 저장
- 클러스터 ↔ 핀 전환 기준: 줌 레벨 14 (동 단위 식별 가능 시점)

### 2.3 유저 프로필

| 필드 | 필수 | 설명 |
|------|------|------|
| 닉네임 | O | 표시 이름 |
| 프로필 사진 | X | 본인 사진 (선택) |
| 지역 | O | 시/구/동 |
| 아이 정보 | O | 아래 상세 |
| 자기소개 | X | 한줄 소개 |

#### 아이 정보 (복수 등록 가능)

| 필드 | 필수 | 설명 |
|------|------|------|
| 태명/별명 | O | 공개용 이름 |
| 생년월 | O | 개월수 자동 계산 |
| 성별 | X | 남/여/비공개 |

### 2.5 채팅 (Firebase 기반)

| 기능 | 설명 |
|------|------|
| 방 채팅 | 방 참여자 간 그룹 채팅 (Firestore 실시간) |
| 1:1 채팅 | 부모 간 개별 대화 |
| 알림 | 새 메시지 → FCM 푸시 알림 |
| 사진 전송 | 이미지 전송 (Firebase Storage) |

- 채팅 데이터는 Firestore에 저장 (실시간 리스너로 메시지 수신)
- NestJS 서버에서 직접 WebSocket 관리하지 않음
- 서버는 Firestore Admin SDK로 채팅방 생성/삭제/권한 관리만 담당

### 2.6 신뢰 시스템

| 기능 | 설명 |
|------|------|
| 본인 인증 | 휴대폰 번호 인증 (필수) — 인증 업체/방식 추후 결정 |
| 모임 후기 | 방 종료 후 참여자 간 후기 작성 |
| 매너 온도 | 후기 기반 신뢰 점수 |
| 신고 | 부적절 유저/방 신고 |
| 차단 | 특정 유저 차단 |

### 2.7 알림

| 알림 유형 | 설명 |
|------------|------|
| 참여 신청 알림 | 내 방에 누가 신청했을 때 (방장) |
| 수락/거절 알림 | 신청 결과 (신청자) |
| 모임 리마인더 | 모임 1시간 전, 하루 전 |
| 채팅 알림 | 새 메시지 |
| 새 방 알림 | 내 조건에 맞는 새 방이 올라왔을 때 |

## 3. 화면 구성 (Screen Flow)

### 3.1 메인 탭 구조

```
[홈] [검색] [방 만들기(+)] [채팅] [마이페이지]
```

### 3.2 주요 화면 목록

| 화면 | 설명 |
|------|------|
| 스플래시 | 앱 로딩 |
| 온보딩 | 서비스 소개 (3페이지) |
| 로그인 | 소셜 로그인 (카카오, Apple, Google) + 이메일 로그인 |
| 회원가입 | 프로필 + 아이 정보 + 핸드폰 본인 인증 |
| 홈 | 내 지역 기반 추천 방 목록 + 지도 뷰 전환 |
| 지도 | 네이버 지도 (클러스터/핀 뷰) |
| 방 상세 | 방 정보 + 참여자 목록 + 참여 신청 |
| 방 생성 | 새 방 만들기 폼 |
| 검색 | 필터 기반 방 검색 |
| 참여 관리 | 방장용 - 신청자 수락/거절 |
| 채팅 목록 | 참여중인 방 채팅 + 1:1 채팅 |
| 채팅방 | 메시지 주고받기 |
| 마이페이지 | 프로필 / 내 모임 / 후기 / 설정 |
| 프로필 수정 | 내 정보 + 아이 정보 수정 |
| 유저 프로필 | 다른 유저 정보 조회 |
| 알림 | 알림 목록 |

### 3.3 유저 플로우

```
[첫 방문]
앱 설치 → 온보딩 → 소셜 로그인 → 프로필 설정 → 아이 정보 입력 → 홈

[방 참여]
홈/검색 → 방 발견 → 방 상세 확인 → 참여 신청 → (자유입장: 즉시 참여 / 승인: 대기) → 채팅방 입장

[방 생성]
+ 버튼 → 방 생성 폼 → 등록 → 신청자 대기 → 수락/거절 → 모임 진행

[모임 완료]
모임 시간 경과 → 방 상태 "완료" → 후기 작성 요청 알림 → 후기 작성
```

## 4. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 플랫폼 | iOS, Android (Flutter 크로스플랫폼) |
| 서버 | NestJS + TypeORM |
| DB | PostgreSQL |
| 실시간 채팅 | Firebase Firestore (실시간 리스너) |
| 인증 | JWT + Refresh Token |
| 소셜 로그인 | 카카오, Apple, Google |
| 이메일 로그인 | 이메일 + 비밀번호 |
| 본인 인증 | 핸드폰 본인 인증 (업체 추후 결정) |
| 푸시 알림 | FCM (Firebase Cloud Messaging) |
| 파일 저장 | Firebase Storage |
| 배포 | AWS EC2/ECS 또는 Railway |

## 5. MVP 범위 (1단계)

> 핵심만 빠르게 만들어서 반응 확인

### MVP 포함

- [x] 소셜 로그인 (카카오, Apple, Google) + 이메일 로그인
- [x] 핸드폰 본인 인증
- [x] 프로필 + 아이 정보 등록
- [x] 방 생성 (지역, 날짜, 시간, 개월수, 장소, 인원, 입장방식)
- [x] 방 목록 조회 (지역/개월수 필터, 약속일 지난 방 자동 제외)
- [x] 네이버 지도 뷰 (클러스터 + 핀)
- [x] 방 상세 보기
- [x] 참여 신청 (자유입장 + 승인)
- [x] 방 채팅 (Firebase Firestore 기반)
- [x] 푸시 알림 (참여 신청, 수락/거절)
- [x] 마이페이지 (내 모임 목록)

- [ ] 신고/차단

- [ ] 번개 모임 (긴급)
- [ ] 육아 크루 (고정 그룹)
